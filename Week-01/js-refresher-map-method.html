JavaScript map() Method

The map() method is an ES5 feature that creates a new array by applying a function to each element of the original array. It skips empty elements and does not modify the original array.

Syntax

arr.map((element, index, array) => { /* … */ })
  • element: Required — the current element being processed
  • index: Optional — the index of the current element
  • array: Optional — the original array being processed

Return Value: A new array with the results of the callback function.

Example 1: Double Each Element

const a = [1, 2, 3, 4];
const b = a.map(x => x * 2);
console.log(b);
[2, 4, 6, 8]

Example 2: Square Roots of Each Number

const a = [1, 4, 9, 16, 25];
const sr = a.map(num => Math.sqrt(num));
console.log(sr);
[1, 2, 3, 4, 5]

Example 3: Return Object with Index and Square

let a = [2, 5, 6, 3, 8, 9];
let res = a.map(function (val, index) {
  return { key: index, value: val * val };
});
console.log(res);
[ { key: 0, value: 4 }, { key: 1, value: 25 }, { key: 2, value: 36 }, { key: 3, value: 9 }, { key: 4, value: 64 }, { key: 5, value: 81 } ]

Example 4: Concatenate 'A' with Each Character of a String

let s = "Geeks";
let res = Array.prototype.map.call(s, function (item) {
  return item + 'A';
});
console.log(res);
['GA', 'eA', 'eA', 'kA', 'sA']

Example 5: Using parseInt() with map()

const a = ['10', '20', '30'];
const b = a.map(s => parseInt(s));
console.log(b);
[10, 20, 30]
