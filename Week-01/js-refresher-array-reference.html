JavaScript Array Reference

JavaScript Arrays are used to store multiple elements in a single variable. They can hold numbers, strings, objects, and even other arrays.

Syntax

const arr = ["Item1", "Item2", "Item3"];

Example: Copying Array Items

// Create and initialize an array
let courses = ["HTML", "CSS", "JavaScript", "React"];

// Display the array items
console.log(courses);

// Create a new empty array
let newArray = [];

// Copy elements using forEach
courses.forEach(function (course) {
    newArray.push(course);
});

// Display the new array
console.log(newArray);
[ 'HTML', 'CSS', 'JavaScript', 'React' ]
[ 'HTML', 'CSS', 'JavaScript', 'React' ]

JavaScript Array Constructor

Constructor Description
Array() Creates a new array.

JavaScript Array Properties

Instance Property Description
constructor Returns a reference to the array function.
length Gets or sets the number of elements in an array.

JavaScript Array Methods

Static Methods

Method Description
Array.from() Creates an array from iterable or array-like objects.
Array.isArray() Returns true if the value is an array.
Array.of() Creates a new array instance with given elements.

Instance Methods

Method Description
at() Returns the element at the specified index.
concat() Merges two or more arrays.
copyWithin() Copies part of the array within itself.
entries() Returns key/value pairs from the array.
every() Tests whether all elements pass a condition.
fill() Fills all the elements with a static value.
filter() Creates a new array with elements that pass the test.
find() Returns the first element that satisfies the condition.
findIndex() Returns the index of the first matching element.
flat() Flattens nested arrays.
flatMap() Maps and flattens the result.
forEach() Executes a function once for each element.
includes() Checks if array contains a specified value.
indexOf() Returns the first index of an element.
join() Joins all elements into a string.
keys() Returns an array iterator with keys.
lastIndexOf() Returns the last index of an element.
map() Creates a new array with the results of calling a function.
pop() Removes the last element and returns it.
push() Adds new elements to the end.
reduce() Applies a function to reduce to a single value.
reduceRight() Applies reduce from right to left.
reverse() Reverses the array in place.
shift() Removes the first element.
slice() Returns a portion of an array.
some() Tests whether at least one element passes the test.
sort() Sorts the array in place.
splice() Changes the contents by removing or replacing.
toLocaleString() Converts elements to a localized string.
toString() Converts array to string.
unshift() Adds new elements to the beginning.
values() Returns an iterator with the array values.
