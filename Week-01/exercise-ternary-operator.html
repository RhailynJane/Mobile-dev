Ternary Operator Exercise

Instruction 1: Age Checker

- Write a function called checkAge that takes a parameter age.
- Use the ternary operator to return "You are an adult." if age is 18 or older, and "You are a minor." if age is less than 18.
- Test the function with different values of age.

Solution:


function checkAge(age) {
  return age >= 18 ? "You are an adult." : "You are a minor.";
}

console.log(checkAge(20));
console.log(checkAge(15));

Instruction 2: Even or Odd

- Write a function called isEven that takes a number as a parameter.
- Use the ternary operator to return "Even" if the number is even, and "Odd" if the number is odd.
- Test the function with different numbers.

Solution:


function isEven(num) {
  return num % 2 === 0 ? "Even" : "Odd";
}

console.log(isEven(4));
console.log(isEven(7));

Instruction 3: Greeting Based on Time

- Write a function called getGreeting that takes a parameter hour.
- Use the ternary operator to return "Good Morning!" if hour is less than 12, "Good Afternoon!" if hour is between 12 and 18 (inclusive), and "Good Evening!" if hour is greater than 18.
- Test the function with different values of hour.

Solution:


function getGreeting(hour) {
  return hour < 12
    ? "Good Morning!"
    : hour <= 18
    ? "Good Afternoon!"
    : "Good Evening!";
}

console.log(getGreeting(10));
console.log(getGreeting(15));
console.log(getGreeting(20));

Instruction 4: Student Pass/Fail

- Write a function called passOrFail that takes a parameter score.
- Use the ternary operator to return "Pass" if score is 50 or more, and "Fail" if score is less than 50.
