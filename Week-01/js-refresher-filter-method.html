JavaScript filter() Method

The filter() method creates a new array containing elements that satisfy a specified condition. It skips empty elements and does not modify the original array.

Syntax

array.filter(callback(element, index, arr), thisValue)

Parameters:

  • callback: Function to test each element
  • element: Current element being processed
  • index (Optional): Index of the current element
  • arr (Optional): Original array
  • thisValue (Optional): Value used as this when executing callback

Return value: A new array with elements that pass the test; returns an empty array if none match.

Example 1: Filter Ages That Can Vote

function canVote(age) {
  return age >= 18;
}

let filtered = [24, 33, 16, 40].filter(canVote);
console.log(filtered);
[24, 33, 40]

Example 2: Filter Positive Numbers

function isPositive(value) {
  return value > 0;
}

let filtered = [112, 52, 0, -1, 944].filter(isPositive);
console.log(filtered);
[112, 52, 944]

Example 3: Filter Even Numbers

function isEven(value) {
  return value % 2 === 0;
}

let filtered = [11, 98, 31, 23, 944].filter(isEven);
console.log(filtered);
[98, 944]
