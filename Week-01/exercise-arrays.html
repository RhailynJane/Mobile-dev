JavaScript Array Exercise

1. Create an Array

Create an array called students with elements: "Alice", "Bob", "Charlie". Then log the length of the array. 

const students = ["Alice", "Bob", "Charlie"];
console.log(students.length);
Output:
3

2. Add an Element to the Array

Use the length property to add a new student "David" to the end of the array. Log the updated array and its length. 

students[students.length] = "David";
console.log(students);
console.log(students.length);
Output:
["Alice", "Bob", "Charlie", "David"]
4

3. Iterate Over the Array

Use a for loop to log each student name to the console.

for (let i = 0; i < students.length; i++) {
  console.log(students[i]);
}
Output:
Alice
Bob
Charlie
David

4. Remove an Element from the Array

Use the length property to remove the last element by reducing the length by 1. 

students.length = students.length - 1;
console.log(students);
console.log(students.length);
Output:
["Alice", "Bob", "Charlie"]
3

5. Empty the Array

Set students.length = 0 to clear all elements.

students.length = 0;
console.log(students);
console.log(students.length);
Output:
[]
0

6. Add Multiple Elements

Use push() to add "Eve", "Frank", and "Grace", then log the array length after each addition. Finally, print each name. 

students.push("Eve");
console.log(students.length);
students.push("Frank");
console.log(students.length);
students.push("Grace");
console.log(students.length);
for (let i = 0; i < students.length; i++) {
  console.log(students[i]);
}
Output:
1
2
3
Eve
Frank
Grace
