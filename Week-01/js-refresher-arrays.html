JavaScript Arrays

In JavaScript, an array is an ordered list of values. Each value is called an element, and each element has a numeric position in the array, known as its index. Arrays in JavaScript are zero-indexed, meaning the first element is at index 0, the second at index 1, and so on.

1. Create Array using Literal

// Creating an Empty Array
let a = [];
console.log(a);

// Creating an Array and Initializing with Values
let b = [10, 20, 30];
console.log(b);
Output:
[]
[10, 20, 30]

2. Create using new Keyword (Constructor)

// Creating and Initializing an array with values
let a = new Array(10, 20, 30);
console.log(a);
Output:
[10, 20, 30]

3. Accessing Elements of an Array

// You can access elements using their index, starting from 0.
const fruits = ["Kiwi", "Apple", "Orange"];
console.log(fruits[0]);
console.log(fruits[1]);
Output:
Kiwi
Apple

4. Accessing the First Element of an Array

// Access the first element using index 0
let fst = fruits[0];
console.log("First Item: ", fst);
Output:
First Item: Kiwi

5. Accessing the Last Element of an Array

// Use [array.length - 1] to access the last element
let lst = fruits[fruits.length - 1];
console.log("Last Item: ", lst);
Output:
Last Item: Orange

6. Modifying the Array Elements

// You can change elements by reassigning a new value to an index
console.log(fruits);
fruits[0] = "Grapes";
console.log(fruits);
Output:
["Kiwi", "Apple", "Orange"]
["Grapes", "Apple", "Orange"]

7. Adding Elements to the Array

// Use push() to add at the end, unshift() to add at the start
fruits.push("Banana");
console.log(fruits);
fruits.unshift("Blueberry");
console.log(fruits);
Output:
["Grapes", "Apple", "Orange", "Banana"]
["Blueberry", "Grapes", "Apple", "Orange", "Banana"]

8. Array Length

// Use length property to get the total number of elements
console.log("Array Length: ", fruits.length);
Output:
Array Length: 5

9. Removing Elements from an Array

// Use pop(), shift(), and splice() to remove elements
console.log("Original Array: " + fruits);
fruits.pop();
console.log("After Removing the last: " + fruits);
fruits.shift();
console.log("After Removing the First: " + fruits);
fruits.splice(1, 2);
console.log("After Removing 2 elements starting from index 1: " + fruits);
Output:
Original Array: Blueberry,Grapes,Apple,Orange,Banana
After Removing the last: Blueberry,Grapes,Apple,Orange
After Removing the First: Grapes,Apple,Orange
After Removing 2 elements starting from index 1: Grapes

10. Array Length Importance

// Knowing length helps in determining total elements and looping through the array
let a = ["HTML", "CSS", "JS"];
console.log(a.length);

const numbers = [1, 2, 3, 4, 5];
for (let i = 0; i < numbers.length; i++) {
  console.log(numbers[i]);
}
Output:
3
1
2
3
4
5

11. Add Element Using Length

// Use length to add an element to the end without push()
const animals = ["dogs", "cats"];
animals[animals.length] = "rabbit";
console.log(animals);
Output:
["dogs", "cats", "rabbit"]

12. Truncating an Array

// You can shorten an array by changing its length
const colors = ["Red", "Green", "Blue", "Yellow"];
colors.length = 2;
console.log(colors);
Output:
["Red", "Green"]

13. Dynamic Nature of JavaScript Arrays

// Arrays grow or shrink as elements are added or removed
const items = [];
items.push("Item1");
items.push("Items2");
console.log(items.length);

let sub = ["HTML", "CSS", "JS"];
sub.length = 7;
console.log("After Increasing Length: ", sub);
sub.length = 2;
console.log("After Decreasing Length: ", sub);
Output:
2
After Increasing Length: ["HTML", "CSS", "JS", , , , ]
After Decreasing Length: ["HTML", "CSS"]

14. Array Concatenation

// Use concat() to join two arrays into a new one
let course1 = ["HTML", "CSS", "JS", "React"];
let course2 = ["Node.js", "Express.js"];
let concateArray = course1.concat(course2);
console.log("Concatenated Array: ", concateArray);
Output:
Concatenated Array: ["HTML", "CSS", "JS", "React", "Node.js", "Express.js"]

15. Array toString()

// Converts the array to a comma-separated string
const soda = ["Coke", "Sprite", "Mountain Dew"];
console.log("Array: ", soda);
console.log("Converted to String: ", soda.toString());
Output:
Array: ["Coke", "Sprite", "Mountain Dew"]
Converted to String: Coke,Sprite,Mountain Dew

16. Array Join

// Similar to toString(), but lets you define a separator
console.log(soda.join("*"));
Output:
Coke*Sprite*Mountain Dew

17. Check the Type of an Array

// Use typeof to check type (returns 'object' for arrays)
console.log(typeof soda);
Output:
object
