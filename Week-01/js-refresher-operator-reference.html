JavaScript Operators

JavaScript Operators are used to perform specific mathematical and logical computations on operands. They are used to compare values, perform arithmetic operations, and more.

Example: Using typeof Operator

let a = 17;
let b = "GeeksforGeeks";
let c = "";
let d = null;

console.log("Type of a = " + (typeof a));
console.log("Type of b = " + (typeof b));
console.log("Type of c = " + (typeof c));
console.log("Type of d = " + (typeof d));
console.log("Type of e = " + (typeof e));
Output: 
Type of a = number
Type of b = string
Type of c = string
Type of d = object
Type of e = undefined

JavaScript Arithmetic Operators

Operator Name Operation
+ Addition Add numbers or concatenate strings
- Subtraction Subtract two numbers
* Multiplication Multiply two numbers
/ Division Divide two numbers
% Modulus Find remainder
** Exponentiation Raise left operand to power of right
++ Increment Increase by one
-- Decrement Decrease by one

JavaScript Comparison Operators

Operator Name Operation
== Equality Check if equal
!= Inequality Check if not equal
=== Strict Equality Check if equal and same type
!== Strict Inequality Check if not equal or not same type
> Greater Than Check if greater
< Less Than Check if smaller
>= Greater or Equal Check if greater or equal
<= Less or Equal Check if smaller or equal

JavaScript Logical Operators

Operator Name Operation
! NOT Returns opposite boolean
&& AND True if all operands are true
|| OR True if at least one operand is true

JavaScript Bitwise Operators

Operator Name Operation
& AND True if both bits are 1
| OR True if at least one bit is 1
^ XOR True if bits are different
~ NOT Inverts the bits
<< Left Shift Shift bits left
>> Right Shift Shift bits right
>>> Zero Fill Right Shift Right shift and fill with 0

JavaScript Assignment Operators

Operator Name Operation
+= Addition Assignment Add and assign
-= Subtraction Assignment Subtract and assign
*= Multiplication Assignment Multiply and assign
/= Division Assignment Divide and assign
%= Remainder Assignment Remainder and assign
**= Exponentiation Assignment Exponent and assign
<<= Left Shift Assignment Left shift and assign
>>= Right Shift Assignment Right shift and assign
&= Bitwise AND Assignment AND and assign
|= Bitwise OR Assignment OR and assign
^= Bitwise XOR Assignment XOR and assign

Few Important JavaScript Operators

  • Ternary (?:) - Simplified if-else
  • typeof - Returns data type
  • instanceof - Checks object type at runtime
  • delete - Removes property from object
  • in - Checks if a property exists in object
  • ... (Spread) - Expands iterable to elements
  • , (Comma) - Separates operands
  • ( ) (Grouping) - Overrides precedence
  • ?? (Nullish Coalescing) - Returns fallback if null/undefined
  • ?. (Optional chaining) - Safe property access
  • |> (Pipeline) - Pipes value into function (stage 2 proposal)
  • => (Arrow) - Short function syntax
