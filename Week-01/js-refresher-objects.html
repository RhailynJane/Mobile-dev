JavaScript Objects

Objects in JavaScript are dynamic data structures used to store related information as key-value pairs. They can be created, modified, and interacted with in various ways.

1. Creation Using Object Literal

This is the most common way to define an object, using curly braces {} with key-value pairs. 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
console.log(user);
Output:
{ name: "Rhailyn", age: 28, job: "Student", greet: ƒ }

2. Creation Using Object Constructor

You can also create objects using the new Object() constructor and assign properties one by one. 

let obj = new Object();
obj.name = "Rhailyn";
obj.age = 28;
obj.job = "Student";
console.log(obj);
Output:
{ name: "Rhailyn", age: 28, job: "Student" }

3. Modifying Object Properties

You can modify the value of an existing property by simply assigning it a new value. 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
user.age = 20;
console.log(user);
Output:
{ name: "Rhailyn", age: 20, job: "Student", greet: ƒ }

4. Adding Properties to an Object

You can add new properties to an existing object using dot notation or bracket notation. 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
user.nationality = "Filipino";
console.log(user);
Output:
{ name: "Rhailyn", age: 28, job: "Student", greet: ƒ, nationality: "Filipino" }

5. Removing Properties from an Object

Use the delete operator to remove a property from an object. 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
user.nationality = "Filipino";
delete user.nationality;
console.log(user);
Output:
{ name: "Rhailyn", age: 28, job: "Student", greet: ƒ }

6. Checking if a Property Exists

You can check whether a property exists using the in operator or hasOwnProperty(). 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
console.log("nationality" in user);
console.log(user.hasOwnProperty("age"));
Output:
false
true

7. Using Methods

Objects can contain functions called methods. Use this keyword to refer to the object itself. 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
user.greet();
Output:
Hello Rhailyn

8. Iterating Through Object Properties

Use a for...in loop to iterate through all enumerable properties in an object. 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
for (let key in user) {
  console.log(key + ": " + user[key]);
}
Output:
name: Rhailyn
age: 28
job: Student
greet: function () { console.log("Hello " + this.name); }

9. Merging Objects

You can merge multiple objects into one using the spread syntax {...obj1, ...obj2}. 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
let user1 = { name: "Rhai" };
let user2 = { age: 28 };
let user3 = { ...user1, ...user2 };
console.log(user3);
Output:
{ name: "Rhai", age: 28 }

10. Object Length

Use Object.keys() to get an array of keys, and .length to find how many properties an object has. 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
console.log(Object.keys(user).length);
Output:
4

11. Recognizing an Object

To confirm a value is an object, use typeof and check it is not null. 

let user = {
  name: "Rhailyn",
  age: 28,
  job: "Student",
  greet: function () {
    console.log("Hello " + this.name);
  },
};
console.log(typeof user === "object" && user !== null);
Output:
true

12. Accessing Object Properties

You can access an object’s properties using either dot notation or bracket notation. 

let obj = { name: "Rhai", age: 28 };

// Using Dot Notation
console.log(obj.name);

// Using Bracket Notation
console.log(obj["age"]);
Output:
Rhai
28
