Object Exercise

Instructions:

  1. Create an Object: Create an object called car with properties: make, model, year, color, and isRunning.
  2. Access and Log Properties: Access and log properties of the car object.
  3. Modify Properties: Change the color property and log it.
  4. Add a Method: Add start method to set isRunning to true and log a message.
  5. Call the Method: Call start and log isRunning.
  6. Add a New Property: Add fuelLevel and log it.
  7. Create a Method to Refuel: Add refuel method to update fuelLevel.
  8. Call the Refuel Method: Call refuel with a parameter and log the result.

JavaScript Code:

// Step 1: Create an Object
let car = {
  make: "Toyota",
  model: "Corolla",
  year: 2021,
  color: "Red",
  isRunning: false
};

// Step 2: Access and Log Properties
console.log(car.make);   // Output: "Toyota"
console.log(car.year);   // Output: 2021

// Step 3: Modify Properties
car.color = "Blue";
console.log(car.color);  // Output: "Blue"

// Step 4: Add a Method
car.start = function() {
  this.isRunning = true;
  console.log("The car has started.");
};

// Step 5: Call the Method
car.start();
console.log(car.isRunning);  // Output: true

// Step 6: Add a New Property
car.fuelLevel = 50;
console.log(car.fuelLevel);  // Output: 50

// Step 7: Create a Method to Refuel
car.refuel = function(amount) {
  this.fuelLevel += amount;
  console.log("Fuel level is now: " + this.fuelLevel);
};

// Step 8: Call the Refuel Method
car.refuel(20);  // Output: "Fuel level is now: 70"
