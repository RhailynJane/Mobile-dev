JavaScript Ternary Operator

The Ternary Operator in JavaScript is a shortcut for writing simple if-else statements. It’s also known as the Conditional Operator because it works based on a condition. The ternary operator allows you to quickly decide between two values depending on whether a condition is true or false.

Syntax:

condition ? expressionIfTrue : expressionIfFalse
  • condition: A condition that evaluates to true or false.
  • expressionIfTrue: The value or expression returned if the condition is true.
  • expressionIfFalse: The value or expression returned if the condition is false.

Example:

let PMarks = 50;

let res = (PMarks > 39) ? "Pass" : "Fail";

console.log(res); // Output: Pass
Output: Pass

Characteristics of Ternary Operator

  • Shorter Syntax: Helps write conditions more concisely than if...else.
  • Evaluates a Condition: Checks if a condition is true or false.
  • Three Parts: The condition, the result if true, and the result if false.
  • Returns a Value: Directly gives a result that can be assigned or used.
  • Can Be Nested: You can nest ternary operators to handle multiple conditions.

Nested Ternary Operators

let day = 3;
let greeting = (day === 1) ? 'Start of the week' :
               (day === 2) ? 'Second day' :
               (day === 3) ? 'Midweek' :
               (day === 4) ? 'Almost weekend' :
               'Weekend';

console.log(greeting); // Output: Midweek
Output: Midweek

Ternary Operator in Functions

function checkAge(age) {
  return (age >= 18) ? 'Adult' : 'Minor';
}

console.log(checkAge(20)); // Output: Adult
console.log(checkAge(15)); // Output: Minor
Output: Adult Minor

Using the Ternary Operator with Function Calls

function sayHello(test) {
  console.log(`Hello, ${test}!`);
}

function sayGoodbye(test) {
  console.log(`Goodbye, ${test}!`);
}

let isLeaving = true;
let test = 'Rhai';

isLeaving ? sayGoodbye(test) : sayHello(test);  // Output: Goodbye, Rhai!
Output: Goodbye, Rhai!

A Comparison of the Ternary Operator and the if...else Statement

Using if...else Statement

let hour = 15;
let message;

if (hour < 12) {
  message = 'Good morning';
} else {
  message = 'Good afternoon';
}

console.log(message); // Output: Good afternoon
Output: Good afternoon

Using the Ternary Operator

let hour = 15;
let message = (hour < 12) ? 'Good morning' : 'Good afternoon';

console.log(message); // Output: Good afternoon
Output: Good afternoon
