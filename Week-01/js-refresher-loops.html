JavaScript Loops

Loops in JavaScript are used to reduce repetitive tasks by repeatedly executing a block of code as long as a specified condition is true. This makes code more concise and efficient.

Example: Print "Hello World" 5 times

for (let i = 0; i < 5; i++) {
  console.log("Hello World!");
}
Hello World!
Hello World!
Hello World!
Hello World!
Hello World!

1. JavaScript for Loop

The for loop repeats a block of code a specific number of times. 

for (let i = 1; i <= 3; i++) {
  console.log("Count:", i);
}
Count: 1
Count: 2
Count: 3

2. JavaScript while Loop

The while loop executes as long as the condition is true. 

let i = 0;
while (i < 3) {
  console.log("Number:", i);
  i++;
}
Number: 0
Number: 1
Number: 2

3. JavaScript do-while Loop

The do-while loop executes the block at least once before checking the condition. 

let i = 0;
do {
  console.log("Iteration:", i);
  i++;
} while (i < 3);
Iteration: 0
Iteration: 1
Iteration: 2

4. JavaScript for-in Loop

The for...in loop iterates over the keys of an object.

const obj = { name: "Ashish", age: 25 };
for (let key in obj) {
  console.log(key, ":", obj[key]);
}
name : Ashish
age : 25

5. JavaScript for-of Loop

The for...of loop iterates over values of iterable objects like arrays. 

let a = [1, 2, 3, 4, 5];
for (let val of a) {
  console.log(val);
}
1
2
3
4
5

6. JavaScript forEach Method

The forEach method executes a provided function once for each array element. 

let nums = [10, 20, 30];
nums.forEach(function(num) {
  console.log("Value:", num);
});
Value: 10
Value: 20
Value: 30

Choosing the Right Loop

  • for loop: Use when the number of iterations is known.
  • while loop: Use when the condition depends on dynamic factors.
  • do-while loop: Use when the loop must run at least once.
  • for...in loop: Use for iterating over object properties.
  • for...of loop: Use for iterable objects like arrays or strings.
  • forEach: Use to process each element in an array with a function.
