JavaScript Spread Operator

The Spread operator (...) is used with iterables like arrays and strings, or object properties, to expand them where multiple elements or properties are expected.

1. Adding Multiple Elements Using Spread Operator


let a = [10, 20];
let b = [...a, 30, 40];

console.log(a); // Output: [10, 20]
console.log(b); // Output: [10, 20, 30, 40]

You can also insert at the beginning and both ends:


let a = [10, 20];
let b = [30, 40, ...a, 50, 60];

console.log(b); // Output: [30, 40, 10, 20, 50, 60]

2. Find Min / Max using Spread Operator


let a = [1, 2, 3, -1];
console.log(Math.min(a));      // NaN
console.log(Math.min(...a));   // -1

3. Passing Array Elements as Function Parameters


function add(x, y, z) {
  return x + y + z;
}

let a = [10, 20, 30];
console.log(add(...a)); // Output: 60

4. Copying Array using Spread


const a = [1, 2, 3];
const b = [...a]; 
console.log(b); // Output: [1, 2, 3]

5. Concatenate Arrays using Spread Operator


let a = [1, 2, 3];
let b = [4, 5];

a = [...a, ...b];
console.log(a); // Output: [1, 2, 3, 4, 5]

6. Working with Objects using Spread Operator


// Cloning an object
const usr = {
  name: 'Jen',
  age: 22
};

const cloneUsr = { ...usr };
console.log(cloneUsr); // Output: { name: 'Jen', age: 22 }
  

// Merging two objects
const usr1 = { name: 'Jen', age: 22 };
const usr2 = { name: 'Andrew', location: 'Philadelphia' };

const mergedUsers = { ...usr1, ...usr2 };
console.log(mergedUsers); // Output: { name: 'Andrew', age: 22, location: 'Philadelphia' }
  

// Adding properties to an object
const o1 = { a: 1, b: 2 };
const o2 = { ...o1, b: 3, c: 4 };
console.log(o2); // Output: { a: 1, b: 3, c: 4 }
