JavaScript Functions

Functions in JavaScript are reusable blocks of code designed to perform specific tasks. They allow you to organize, reuse, and modularize code. It can take inputs, perform actions, and return outputs.

Example 1: Basic Function

function sum(x, y) {
    return x + y;
}
console.log(sum(6, 9));

Output:
15

Function Syntax and Working

  • Begin with the keyword function.
  • A user-defined function name.
  • List of parameters inside parentheses.
  • Function body enclosed in curly braces {}.

Return Statement

Use return to return a value from a function. This is optional but often used.

Function Parameters

These are the inputs to a function. For example: sum(x, y)

Calling Functions

// Function Definition
function welcomeMsg(name) {
    return ("Hello " + name + " welcome to GeeksforGeeks");
}

let nameVal = "User";

// Function Call
console.log(welcomeMsg(nameVal));

Output:
Hello User welcome to GeeksforGeeks

Function Invocation

Function code executes when called, triggered by events, manually, or automatically.

Function Expression

const mul = function(x, y) {
    return x * y;
};
console.log(mul(4, 5));

Output:
20

Arrow Functions

const a = ["Hydrogen", "Helium", "Lithium", "Beryllium"];

const a2 = a.map(function (s) {
    return s.length;
});
console.log("Normal way", a2);

const a3 = a.map((s) => s.length);
console.log("Using Arrow Function", a3);

Output:
Normal way [8, 6, 7, 9]
Using Arrow Function [8, 6, 7, 9]

Immediately Invoked Function Expression (IIFE)

(function () {
    console.log("This runs immediately!");
})();

Output:
This runs immediately!

Callback Functions

function num(n, callback) {
    return callback(n);
}

const double = (n) => n * 2;
console.log(num(5, double));

Output:
10

Anonymous Functions

setTimeout(function () {
    console.log("Anonymous function executed!");
}, 1000);

Nested Functions

function outerFun(a) {
    function innerFun(b) {
        return a + b;
    }
    return innerFun;
}

const addTen = outerFun(10);
console.log(addTen(5));

Output:
15

Pure Functions

function pureAdd(a, b) {
    return a + b;
}
console.log(pureAdd(2, 3));

Output:
5

Key Characteristics of Functions

  • Parameters and Arguments: Functions can accept parameters and be called with arguments.
  • Return Values: Functions can return a value using the return keyword.
  • Default Parameters: Default values can be assigned to function parameters.

Advantages of Functions in JavaScript

  • Reusability: Write code once and use it multiple times.
  • Modularity: Break complex problems into smaller, manageable pieces.
  • Improved Readability: Functions make code easier to understand.
  • Maintainability: Changes can be made in one place without affecting the entire codebase.
